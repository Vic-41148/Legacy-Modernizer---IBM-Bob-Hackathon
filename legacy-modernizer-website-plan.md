## 2. Design Concept: "Modern Tech Innovation"

### Design Philosophy
**Clean, Professional, Tech-Forward**
- Emphasize innovation and technical excellence
- Use data visualization to show transformation results
- Interactive elements to demonstrate the solution
- Code snippets to showcase technical depth

### Color Scheme

**Primary Palette:**
- **IBM Blue:** `#0F62FE` (Primary brand color, trust, technology)
- **Deep Navy:** `#161616` (Backgrounds, sophistication)
- **White:** `#FFFFFF` (Clean, modern)

**Accent Colors:**
- **Success Green:** `#24A148` (Positive results, improvements)
- **Warning Orange:** `#FF832B` (Legacy system indicators)
- **Gradient:** Blue to Purple `#0F62FE → #8A3FFC` (Innovation, transformation)

**Semantic Colors:**
- Legacy/Before: `#FF832B` (Orange - outdated)
- Modern/After: `#24A148` (Green - improved)
- Neutral: `#525252` (Gray - text)

### Typography

**Headings:** IBM Plex Sans (Bold, 700)
- H1: 48px (mobile: 32px)
- H2: 36px (mobile: 28px)
- H3: 28px (mobile: 24px)

**Body:** IBM Plex Sans (Regular, 400)
- Body: 16px / 1.6 line-height
- Small: 14px

**Code:** IBM Plex Mono
- Code blocks: 14px
- Inline code: 14px with background

### Visual Style
- **Layout:** Asymmetric grid with strategic white space
- **Cards:** Subtle shadows, rounded corners (8px)
- **Buttons:** Bold, high contrast, clear CTAs
- **Icons:** Line-style, consistent stroke width
- **Images:** High-quality screenshots, architecture diagrams
- **Animations:** Subtle, purposeful (fade-ins, slide-ups)

### 6.2 Code Comparison Component

```jsx
// CodeComparison.jsx
import { useState } from 'react';
import { Prism as SyntaxHighlighter } from 'react-syntax-highlighter';
import { vscDarkPlus } from 'react-syntax-highlighter/dist/esm/styles/prism';

export default function CodeComparison() {
  const [view, setView] = useState('before');

  const beforeCode = `// Spring 4.3 (Legacy)
@Configuration
@EnableWebMvc
public class WebConfig extends WebMvcConfigurerAdapter {
    @Bean
    public ViewResolver viewResolver() {
        InternalResourceViewResolver resolver = 
            new InternalResourceViewResolver();
        resolver.setPrefix("/WEB-INF/views/");
        resolver.setSuffix(".jsp");
        return resolver;
    }
}`;

  const afterCode = `// Spring Boot 3.2 (Modern)
@SpringBootApplication
public class PetClinicApplication {
    public static void main(String[] args) {
        SpringApplication.run(
            PetClinicApplication.class, args
        );
    }
}
// Auto-configuration handles view resolution`;

  return (
    <div>
      <div className="flex gap-2 mb-4">
        <button
          onClick={() => setView('before')}
          className={`px-4 py-2 rounded ${
            view === 'before' 
              ? 'bg-orange-500 text-white' 
              : 'bg-gray-700 text-gray-300'
          }`}
        >
          Before (Legacy)
        </button>
        <button
          onClick={() => setView('after')}
          className={`px-4 py-2 rounded ${
            view === 'after' 
              ? 'bg-green-500 text-white' 
              : 'bg-gray-700 text-gray-300'
          }`}
        >
          After (Modern)
        </button>
      </div>
      <SyntaxHighlighter 
        language="java" 
        style={vscDarkPlus}
        customStyle={{ borderRadius: '8px', fontSize: '14px' }}
      >
        {view === 'before' ? beforeCode : afterCode}
      </SyntaxHighlighter>
    </div>
  );
}
```
